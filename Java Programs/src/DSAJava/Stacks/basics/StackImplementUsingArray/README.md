***📚 Stack Implementation in Java (Array Based)***
---
This repository contains the implementation of Stack Data Structure using:

✅ Static Array

✅ Dynamic Array

***🧠 What I Learned
How Stack follows LIFO (Last In First Out)
Difference between static vs dynamic array
How resizing works internally in dynamic structures
Better understanding of basic stack operations***
----
***⚙️ Implementations***
---
***🔹 1. Stack Using Static Array
Fixed size defined at start
Fast and simple
Throws overflow when stack is full***
---
***🔹 2. Stack Using Dynamic Array
Automatically resizes when full
More flexible than static stack
Slight extra cost due to resizing***
---
🔑 Operations Implemented
-

***push() → Insert element***

***pop() → Remove top element***

***peek() → Get top element***

***isEmpty() → Check if empty***

***isFull() → (only for static)***
